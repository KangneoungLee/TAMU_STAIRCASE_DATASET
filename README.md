# TAMU_STAIRCASE_DATASET
TAMU staircase dataset for low height UGV

- RGB-D camera parameters

- Image height : 480
- Image width : 848
- Depth scale : 0.001
- Depth type : U16C1
- fx : 425 
- fy : 425
- px : 423
- py :239
